HTML : La Structure du Web

HTML est le fondement de chaque page web. Il sert à structurer le contenu en utilisant des balises comme les titres, les paragraphes, les liens, etc.

Exemple de code HTML :

<html>
  <head>
    <title>Ma Page Web</title>
  </head>
  <body>
    <h1>Bienvenue sur ma page</h1>
    <p>Ceci est un exemple simple</p>
  </body>
</html>
Apprendre HTML

CSS : Mise en Forme du Web

CSS est utilisé pour styliser vos pages web. Vous pouvez changer les couleurs, les tailles de police, l'agencement des éléments, et bien plus.

Exemple de code CSS :

body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
}

h1 {
  color: #3498db;
}
Apprendre CSS

JavaScript : Interactivité sur le Web

JavaScript permet d'ajouter de l'interactivité aux pages web. Vous pouvez créer des animations, valider des formulaires, et bien plus encore.

Exemple de code JavaScript :

let message = "Bonjour, monde!";
alert(message);
Apprendre JavaScript

Les Frameworks JavaScript

Les frameworks JavaScript facilitent le développement d'applications web modernes. Découvrez React, Angular et Vue.js à travers des exemples de code.

React

React est une bibliothèque développée par Facebook pour créer des interfaces utilisateur dynamiques.

Exemple de code :

import React from 'react';
    
                    function App() {
                      return (
                        <div>
                          <h1>Bonjour, React!</h1>
                        </div>
                      );
                    }
                    
                    export default App;

Angular

Angular est un framework complet développé par Google pour des applications web robustes.

Exemple de code :

@Component({
                    selector: 'app-root',
                    template: '<h1>Bonjour, Angular!</h1>'
                  })
                  export class AppComponent { }

Vue.js

Vue.js est un framework progressif pour la création d'interfaces utilisateurs interactives et dynamiques.

Exemple de code :

<template>
                    <h1>Bonjour, Vue.js!</h1>
                  </template>
                  
                  <script>
                  export default {
                    name: 'App',
                  }
                  </script>
